## About

Estudante de **Engenharia de Software** pela UniAmérica, com formação técnica pelo **SENAI**. Transito com naturalidade entre backend, frontend e sistemas embarcados — de APIs REST com Spring Boot a projetos com microcontroladores ESP32.

Experiência profissional como **Instrutor e Operador de Suporte de Alto Valor** na Tel Telemática. Atualmente focado em arquiteturas de software, desenvolvimento full stack e boas práticas de engenharia.
